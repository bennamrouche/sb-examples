# SB-Examples

# Spring Boot Learning Repository

A collection of Spring Boot example applications demonstrating various technologies and best practices.  

This repository is for learning purposes. You are welcome to correct mistakes or improve any code. It is **not** a professional-grade repository.  

Topics covered include:
- JPA (Postgres, MongoDB)
- Security
- Bean management
- REST APIs
- And more

## Getting Started

1. Clone the repository:  
```bash
git clone https://github.com/bennamrouche/sb-examples.git
```

2. Open a project in your favorite IDE (IntelliJ IDEA, Eclipse, VS Code).
3. Configure the database (if applicable) in `application.properties` or `application.yml`.
4. Run the application with:

```bash
   cd sb-jpa-postgres  
    mvn  spring-boot:run
```
